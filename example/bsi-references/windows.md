## Practical takeaway

BSI's current Windows recommendation stack is not a separate 2025 or 2026 Windows guide. It is the Windows client baustein from Edition 2023, updated by the 2025 errata/checklist layer, with the explicit 2023 extension from Windows 10 to Windows 11.
