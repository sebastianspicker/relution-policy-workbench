What’s new in Android Enterprise: Feature Drop
10 June 2025
To meet the rapidly transforming demands of today's workforce, Android Enterprise’s latest Feature Drop provides modern experiences for today’s teams, resilient security and integrated defense, and scalable control with simplified provisioning and governance—all crucial for securing the future of work in the AI era.
Expand all | Collapse all
Resilient security and integrated defense
Android Enterprise’s new multi-layered defenses protect your organization from evolving threats.
Advanced Protection
By strengthening security settings, Advanced Protection enhances the security of Android devices for high-risk users. This helps prevent credential theft and unauthorized access to sensitive company information, offering defense against advanced threats.
Note: Available on Android 16+. Some features will launch later this year; availability of some features may be limited to select Android devices and require individual user activation. Ability for an IT admin to configure is coming soon.
Identity Check
To enhance security, Identity Check requires biometric authentication for accessing critical resources on devices outside designated trusted locations. This added layer of device-level security helps prevent account takeovers and safeguards sensitive information from theft.
Note: Available on Android 16+ devices, Pixel devices running Android 15, and Samsung One UI 7 devices. This requires individual user activation, and cannot be centrally enforced by IT admins.
Corporate badges in Google Wallet
Google Wallet enables employees to use their Android phones as digital corporate ID badges for secure, NFC-based access to workplace buildings. This streamlines employee entry and improves organizational security by centralizing credential management.
Modern experiences built for today’s teams
Android Enterprise provides new intuitive experiences and seamless workflows across powerful hardware.
Material 3 Expressive
Material 3 Expressive updates Android UI’s visual language with customizable looks (colors, fonts, app aesthetics), smoother animations for interactions, better haptic feedback, improved usability for controls and information, and system-wide application across Android, Wear OS, and Google apps.
Notification auto-grouping
Notification auto-grouping combines multiple notifications from a single app into one, reducing clutter on managed and unmanaged devices. This supports employee digital well-being by minimizing digital strain and distraction.
Desktop windowing
Desktop windowing allows users to resize and reposition multiple app windows on tablets, enhancing data security by enabling complex tasks in a secure environment and reducing reliance on less controlled devices.
Taskbar overflow
Taskbar overflow provides an expandable panel for employees to organize and access critical applications, including pinning frequently used apps. This consistent user experience reduces IT support and training needs.
Customized keyboard shortcuts
Customized keyboard shortcuts allow employees to personalize their workflows, increasing user satisfaction and potentially reducing IT support tickets by enabling a more efficient digital experience for IT Decision Makers (ITDMs).
Gemini in Google Docs
Gemini is directly integrated into Google Docs on Android, allowing users to summarize documents, find key information, and draft content directly within the app.
Note: This feature is rolling out to eligible Google Workspace customers with specific Gemini add-ons on Android 8+ phones and tablets. Rollout pace may vary.
PDF on Chrome
PDF on Chrome simplifies document handling by allowing direct PDF viewing within the browser, eliminating the need for separate apps and additional steps.
Scalable control with simplified provisioning and governance
These new Android Enterprise features enhance operational efficiency within your organization through streamlined provisioning and consistent governance.
Efficient device setup
Managed Android devices can be deployed faster and more reliably with fewer setup steps, saving IT resources. Employees gain quicker, smoother access to company resources, reducing support requests and increasing productivity.
Note: Available on Android 16+. For devices running on Android 15, reach out to your OEM for availability.
Device EID access for eSIM management
This feature streamlines eSIM provisioning for managed devices by allowing EMM vendors to read the device's unique Embedded Identity Document (EID). This enables ITDMs to export EID values to carrier partners for scalable eSIM provisioning on deployed devices.
Note: For all devices, this feature is conducted via EMMs. For BYOD devices, the owner activates and can delete the eSIM.
Access Point Name (APN) overrides and 5G slicing via AMAPI
The Android Management API (AMAPI) now allows ITDMs to implement APN overrides for granular control over Android devices’ cellular connections, and to use 5G slicing to route business app traffic through dedicated network slices. APN overrides help enforce security, manage data costs, and restrict access to approved networks, while 5G slicing ensures optimized bandwidth and reliable performance for critical enterprise apps, even under varying network conditions.
Support for Android App Bundle (AAB) files for private apps
Managed Google Play now supports AAB for private apps, allowing IT admins to efficiently deploy, update, and manage proprietary tools, streamlining app delivery and optimizing installation on company devices.
For more info, read Publish private apps from managed Play in your EMM console and Best practices for private apps .
Zero-touch customer portal improvements
The zero-touch customer portal now includes improvements that offer IT admins enhanced control and visibility for more efficient monitoring of account activity.
Detailed audit logs track all provisioning actions.
Customizable role-based permissions enable specific access levels for detailed control over user privileges and enrolled devices.
For more info, read Zero-touch enrollment for IT admins .
